Palette

  • Branching - Palette Branching

Description

This block is used to connect events ports. For more information on how it works please refer to the documentation of the GOTO block by clicking on the link in the "See also" field.

Boîte de dialogue

  • Tag
    This parameter identifies the Goto block whose scope is defined in this block.
    Propriétés : Type 'str' de taille -1.

  • Tag Visibility (1=Local 2=Scoped 3=Global)
    This parameter idetifies the visibility of the block. It can be local(1), scoped(2) or global(3).
    Propriétés : Type 'vec' de taille 1.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: non
  • détection de passage à zéro: non
  • mode: non
  • nombre des entrées évènementielles: 1
  • nombre des sorties évènementielles: 0
  • possède un état continu: non
  • possède un état discret: non
  • possède un état objet: non
  • nom de la fonction de calcul: clkgoto

Fonction d'interface

  • SCI/macros/scicos_blocks/Branching/CLKGOTO.sci [voir code]
