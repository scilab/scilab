set_blockerror - Définit le numéro d'erreur de bloc

Module

  • Scicos

Description

Cette fonction permet de définir une erreur spécifique en cours de simulation pour le bloc courant. Si c'est fonction est utilisée, alors à l'issue de l'éxecution de la fonction de calcul du bloc, le simulateur s'arrêtera et retournera un message d'erreur associé au numéro donné en argument.

Séquence d'appel 

set_blockerror(n)

Paramètres

  • n : un numéro d'erreur. Les appels suivant sont autorisés :
    • set_blockerror(-1)
      le bloc a été appelé avec une valeur d'entrée en dehors du domaine autorisé
    • set_blockerror(-2)
      une singularité a été rencontrée dans le bloc
    • set_blockerror(-3)
      le bloc a produit une erreur interne
    • set_blockerror(-16)
      le bloc ne peut pas allouer de mémoire
