Palette

  • ThermoHydraulics.cosf - Boîte à outils des composants thermohydrauliques

Description

Ce composant représente un puit thermo-hydraulique de pression constante. Ce bloc est défini avec sa pression et sa température. La direction positive est quand le fluide coule dans le bloc.

Boîte de dialogue

  • Pression de la source : P0 (Pa)
    Pression de la source
    Propriétés : Type 'vec' de taille -1

  • Temperature de la source : T0 (K)
    Température de la source
    Propriétés : Type 'vec' de taille -1

  • Enthalpie spécifique de la source : H0 (J/kg)
    Enthalpie spécifique de la source
    Propriétés : Type 'vec' de taille -1

  • 1:température fixée - 2:enthalpie fixée : option_temperature
    Option de temperature. 1:température fixée, 2:enthalpie fixée
    Propriétés : Type 'vec' de taille -1

Propriétés par défaut

  • Entrées :
    • Nom de la variable Modelica : 'C'
      Variable implicite.

  • Paramètres :
    • Nom du paramètre Modelica : 'P0'
      Valeur par défaut : 100000
      Variable d'état : non.

    • Nom du paramètre Modelica : 'T0'
      Valeur par défaut : 290
      Variable d'état : non.

    • Nom du paramètre Modelica : 'option_temperature'
      Valeur par défaut : 1
      Variable d'état : non.
