Palette

Description

Ce composant représente une résistance ohmique variable. La résistance est commandée par le port d'entrée explicite.

Propriétés par défaut

  • Entrées :
    • Nom de la variable Modelica : 'p'
      Variable implicite.

    • Nom de la variable Modelica : 'R'
      Variable explicite.

  • Sorties :
    • Nom de la variable Modelica : 'n'
      Variable implicite.

  • Nom du fichier model : VariableResistor
