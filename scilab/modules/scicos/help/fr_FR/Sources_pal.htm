Palette Sources

Module

  • Scicos

Description

La plupart des blocs de la palette source peuvent être compris comme des générateurs de données. Cette palette contient aussi des blocs pour lire des données dans des fichers, ainsi que des ports d'entrée utilisés dans les superblocs.
Les blocs de cette palette ne possèdent pas de ports réguliers d'entrée.

Blocs

  • CLKINV_f - Port d'entrée événementiel
  • CLOCK_c - Horloge d'activation
  • CONST_m - Constante
  • Counter - Compteur
  • CURV_f - Courbe
  • FROMWSB - Données de l'environnement Scilab vers Scicos
  • GENSIN_f - Générateur sinusoïdal
  • GENSQR_f - Generator de signaux carré
  • IN_f - Port d'entrée régulier
  • INIMPL_f - Port implicie d'entrée
  • Modulo_Count - Compteur modulo
  • RAMP - Rampe
  • RAND_m - Générateur aléatoire
  • READAU_f - Lecteur d'un fichier son AU
  • READC_f - Lecture de donée binaire
  • RFILE_f - Lecture à partir d'un fichier
  • SampleCLK - Horloge synchrone
  • SAWTOOTH_f - Générateur de dents de scie
  • Sigbuilder - Construction de signal
  • STEP_FUNCTION - Générateur de marche d'escalier
  • TIME_f - Temps
  • TKSCALE - Ajuste une valeur constante par un widget tk
