phase_simulation - Retourne la phase actuelle de simulation

Module

  • Scicos

Description

Cette fonction permet de savoir si le simulateur scicos est dans sa phase d'intégration temporelle.

Séquence d'appel 

[psim]=phase_simulation()

Paramètres

  • psim : retourne la phase courante de la simulation.
    • 1 : Le simulateur est sur un point temporel d'activation discrète.
    • 2 : Le simulateur est entrain de réaliser une intégration temporelle à temps continu.
