Palette

  • Sources - Palette Sources

Description

Ce bloc doit seulement être utilisé à l'intérieur des superblocs. Il est utilisé pour représenter un port d'entrée événementiel. Dans un superbloc, les ports doivent être numérotés à partir de 1 jusqu'au nombre de ports.

Boîte de dialogue

  • Port number
    un entier qui définit le numéro de port.
    Propriétés : Type 'vec' de taille 1.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: non
  • détection de passage à zéro: non
  • mode: non
  • nombre des entrées évènementielles: 0
  • nombre des sorties évènementielles: 1
  • possède un état continu: non
  • possède un état discret: non
  • possède un état objet: non
  • nom de la fonction de calcul: input

Fonction d'interface

  • SCI/macros/scicos_blocks/Sources/CLKINV_f.sci [voir code]
