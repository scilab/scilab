Description

The Signal Builder block is a superblock containing a CURVE_c block whose output event port is connected to its input event port. This event feedback gives the possibility to generate events at discontinuous point of the signal. The generated events automatically restart the numerical solver and avoids numerical problems. The generated event is also made available to the user for possible use. Remind that if higher interpolation methods are used, the events are generated only at the beginning and at the end of the signal.

Boîte de dialogue

Les paramètres du bloc Sigbuilder sont identiques à ceux du bloc CURVE_c.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: non
  • détection de passage à zéro: non
  • mode: non
  • sorties régulières:
    - port 1 : taille [-1,1] / type 1
  • nombre des entrées évènementielles: 0
  • nombre des sorties évènementielles: 1
  • possède un état continu: non
  • possède un état discret: non
  • possède un état objet: non
  • nom de la fonction de calcul: csuper
