Palette

  • Sinks - Palette Affichage

Description

Les ports de sortie OUTIMPL_f sont des liens qui permettent de connecter l'extérieur d'un système vers l'intérieur.

Boîte de dialogue

  • Port number
    Spécifie le numéro du port de sortie.
    Propriétés : Type 'vec' de taille 1.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: non
  • détection de passage à zéro: non
  • mode: non
  • entrée régulières:
    - port 1 : taille [-1,1] / type 1
  • nombre des entrées évènementielles: 0
  • nombre des sorties évènementielles: 0
  • possède un état continu: non
  • possède un état discret: non
  • possède un état objet: non
  • nom de la fonction de calcul: outimpl

Fonction d'interface

  • SCI/macros/scicos_blocks/Sinks/OUTIMPL_f.sci [voir code]
