Palette

  • Non_linear - Palette Non-linéaire

Description

Le bloc EXPRESSION applique à son entrée les fonctions Scilab spécifiées en paramètre.

Boîte de dialogue

  • number of inputs
    Un scalaire à valeur entière donnant le nombre de port d'entrée régulier. Ces entrées peuvent-être scalaires ou vectorielles.
    Type 'vec' de taille 1.

  • scilab expression
    Chaine de caractères définissant la relation mathématique à appliquer aux entrées. C'est une expression scilab.
    Type 'str' de taille 1.

  • use zero-crossing (0: no, 1 yes)
    A sélectionner pour permettre la détection de passages à zéro.
    Type 'vec' de taille 1.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: oui
  • détection de passage à zéro: oui
  • mode: oui
  • entrée régulières:
    - port 1 : taille [1,1] / type 1
    - port 2 : taille [1,1] / type 1
  • sorties régulières:
    - port 1 : taille [1,1] / type 1
  • nombre des entrées évènementielles: 0
  • nombre des sorties évènementielles: 0
  • possède un état continu: non
  • possède un état discret: non
  • possède un état objet: non
  • nom de la fonction de calcul: evaluate_expr
