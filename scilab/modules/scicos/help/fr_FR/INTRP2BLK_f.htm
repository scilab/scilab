Palette

  • Non_linear - Palette Non-linéaire

Description

La sortie de ce bloc est fonction de l'entrée obtenue par interpolation bilinéaire. Ce bloc a deux entrées scalaires et une seule sortie scalaire. Les grandeurs et donnent respectivement les coordonnées du point de donnée et sa valeur.

Boîte de dialogue

  • X coord.
    un vecteur de taille n (strictement croissant).
    Propriétés : Type 'vec' de taille -1

  • Y coord.
    un vecteur de taille m (strictement croissant).
    Propriétés : Type 'vec' de taille -1

  • Z values
    une matrice de taille m,n.
    Propriétés : Type 'mat' de taille [-1,-1]

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: oui
  • détection de passage à zéro: non
  • mode: non
  • entrée régulières:
    - port 1 : taille [1,1] / type 1
    - port 2 : taille [1,1] / type 1
  • sorties régulières:
    - port 1 : taille [1,1] / type 1
  • nombre des entrées évènementielles: 0
  • nombre des sorties évènementielles: 0
  • possède un état continu: non
  • possède un état discret: non
  • possède un état objet: non
  • nom de la fonction de calcul: intrp2

Fonction d'interface

  • SCI/macros/scicos_blocks/NonLinear/INTRP2BLK_f.sci [voir code]

Fonction de calcul

  • SCI/routines/scicos/intrp2.f (Type 1) [voir code]
