Palette

  • Electrical.cosf - Boîte à outils des composants électriques

Description

Ce composant est une source de tension CA avec une tension de sortie sinusoidale. L'amplitude et la fréquence de la tension de sortie sont définies par l'utilisateur. La résistance ohmique de ce bloc est zéro.

Boîte de dialogue

  • Amplitude (Volt)
    Amplitude de la tension sinosoidale
    Propriétés : Type 'vec' de taille -1

  • Frequency (Hz)
    Frequency de la tension sinosoidale
    Propriétés : Type 'vec' de taille -1

Propriétés par défaut

  • Entrées :
    • Nom de la variable Modelica : 'p'
      Variable implicite.

  • Sorties :
    • Nom de la variable Modelica : 'n'
      Variable implicite.

  • Paramètres :
    • Nom du paramètre Modelica : 'VA'
      Valeur par défaut : 220
      Variable d'état : non.

    • Nom du paramètre Modelica : 'f'
      Valeur par défaut : 50
      Variable d'état : non.

  • Nom du fichier model : VsourceAC
