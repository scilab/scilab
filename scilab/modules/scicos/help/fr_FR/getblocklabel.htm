getblocklabel - Retourne le label d'un bloc scicos

Module

  • Scicos

Séquence d'appel 

[label]=getblocklabel(blk)

Paramètres

  • blk : Paramètre entier. Définit l'index d'un bloc dans la structure compilée.
  • label : Chaînes de caractères. Retourne le label du bloc numéroté blk.
