Palette

  • Electrical.cosf - Boîte à outils des composants électriques

Description

Ce bloc est un modèle pour une source de tension CA contrôlable. Ce composant fournit une tension sinusoidale entre ses ports. L'amplitude de la tension de sortie est définie par une entrée explicite et la fréquence est définie par l'utilisateur. La résistance ohmique de ce bloc est zéro.

Boîte de dialogue

  • Frequency (Hz)
    Frequency (Hz)
    Propriétés : Type 'vec' de taille -1

Propriétés par défaut

  • Entrées :
    • Nom de la variable Modelica : 'p'
      Variable implicite.

    • Nom de la variable Modelica : 'VA'
      Variable explicite.

  • Sorties :
    • Nom de la variable Modelica : 'n'
      Variable implicite.

  • Paramètres :
    • Nom du paramètre Modelica : 'f'
      Valeur par défaut : 50
      Variable d'état : non.

  • Nom du fichier model : VVsourceAC
