Description

This block is an ideal voltage source. The voltage value is controlled through the explicit input of the block (connected to standard Scicos blocks). The current passing through the block is independent of the voltage across the block terminals.

Default properties

  • Inputs :
    • Modelica variable name : 'vin'
      Explicit variable.

    • Modelica variable name : 'p'
      Implicit variable.

  • Outputs :
    • Modelica variable name : 'n'
      Implicit variable.

  • File name of the model : CVS
