Keyboard_shortcuts - List of the keyboard shortcuts

Customizable keyboard shortcuts :

  • 'a' : Align
  • 'd' : Delete
  • 'c' : Duplicate
  • 'm' : Move
  • 'z' : Smart Move
  • 'u' : Undo
  • 'f' : Flip
  • 't' : Rotate Left
  • 'T' : Rotate Right
  • 'o' : Open/Set
  • 's' : Save
  • 'i' : Get Info
  • 'r' : Replot
  • 'l' : Link
  • 'q' : Quit
  • '-' : Zoom out
  • '+' : Zoom in
  • '=' : Fit diagram to figure
  • 'g' : Grid
  • 'h' : Help

Permanent keyboard shortcuts :

  • 'Ctrl + c' : Copy the selected items in the scicos clipboard
  • 'Ctrl + v' : Paste items from the scicos clipboard
  • 'Ctrl + a' : Select all items of the current window
  • 'Ctrl + s' : Save the current diagram
  • 'Ctrl + left click' : Add items to the current selection
