init_param - add short decription here

Calling Sequence

ga_list = init_param

Parameters

  • ga_list : an initialized list of parameters (this list is empty and is of type plist).

Description

This function initialize an empty list of parameters. You must initialize each of the list of parameters before using them.

Examples

   mylist = init_param();
   mylist = add_param(mylist,'minbound',[0 0 0]);
