add_param - Add a parameter to a list of parameters

Calling Sequence

[ga_list,err] = add_param(list_name,param_name,param_value)

Parameters

  • list_name : the list of parameters. This list must have been initialize by a call to init_param.
  • param_name : a string. The name of the parameter to be added in the list of parameters.
  • param_value : the value associated to the parameter param_name. This parameter is optional. You can set the value of this parameter via a call to set_param.
  • ga_list : the updated list of parameters.
  • err : an error flag which is set to %T if list_name is not of type plist (this list hasn't been initialized by a call to init_param).

Description

This function creates a new parameter in a list of parameters. You can set the value of the parameter using this function or you can set it via a call to set_param.

Examples

   mylist = init_param();
   mylist = add_param(mylist,'minbound',[0 0 0]);
