Threshold palette

Module

  • Scicos

Description

Threshold blocks are used to detect values crossing of state variables during the simulation. This blocks use the solvers (ODE or DAE) to do that operation.

Blocks

  • NEGTOPOS_f - Threshold negative to positive
  • POSTONEG_f - Threshold positive to negative
  • ZCROSS_f - Threshold detection at zero
