Description

This block convert a scalar value into a vector of identical elements.When using blocks with more than one input port (such as the Sum or Relational Operator block), you can mix vector and scalar inputs. When you do this, the scalar inputs are expanded into vectors of identical elements whose widths are equal to the width of the vector inputs. If more than one input port is a vector, they must have the same number of elements.
