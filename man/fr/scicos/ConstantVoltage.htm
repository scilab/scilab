Description

Ce composant est un modèle pour n'importe quel dispositif ou système qui produit une force électromotrice constante entre ses terminaux. La tension aux bornes de cette source est C. C est définie par l'utilisateur. Le port noir indique la tension positive. La résistance ohmique de ce composant est zéro.

Boîte de dialogue

Set ConstantVoltage block parameter  
V (volt) 0.01

  • V (volt) : Type 'vec' de taille 1. Tension de source

  • nombre/taille des sorties sorties régulières: 1 / 1
  • nombre/taille des entrées évènementielles: 0 /
  • nombre/taille des sorties évènementielles: 0 /
  • possède un état continu: non
  • possède un état discret: non
  • nom de la fonction modelica: ConstantVoltage

Fonction d'interface

ConstantVoltage.sci

Modèle Modelica 


class ConstantVoltage "Source for constant voltage"
   Pin p, n;
   parameter Real V = 1 "Volts";
equation
   V = p.v - n.v;
   p.i + n.i = 0;
end ConstantVoltage;
