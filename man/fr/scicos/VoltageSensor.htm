Description

Ce composant est employé pour mesurer la différence de tension entre deux noeuds dans un circuit électrique. Le signal de sortie est la différence entre la tension du port noir et celle du port blanc. La conductance ohmique de ce bloc est zéro.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: oui
  • détection de passage à zéro: non
  • mode: non
  • nombre/taille des entrées régulières: 1 / 1
  • nombre/taille des sorties sorties régulières: 2 / 1 1
  • nombre/taille des entrées évènementielles: 0 /
  • nombre/taille des sorties évènementielles: 0 /
  • possède un état continu: non
  • possède un état discret: non
  • nom de la fonction modelica: VoltageSensor

Fonction d'interface

VoltageSensor.sci

Modèle Modelica 


 class VoltageSensor 
  Pin p;
  Pin n;
  Real v;
 equation
    p.i = 0;
    n.i = 0;
    v = p.v - n.v;
 end VoltageSensor;
