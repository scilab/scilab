Palette

  • Electrical.cosf - Boîte à outils des composants électriques

Description

Ce bloc est inséré en série dans un circuit électrique pour mesurer le courant qui passe à travers. Si le courant positif entre par le port noir, il est consideré positif. La résistance ohmique de ce composant est zéro.

Propriétés par défaut

  • Entrées :
    • Nom de la variable Modelica : 'p'
      Variable implicite.

  • Sorties :
    • Nom de la variable Modelica : 'n'
      Variable implicite.

    • Nom de la variable Modelica : 'i'
      Variable explicite.

  • Nom du fichier model : CurrentSensor
