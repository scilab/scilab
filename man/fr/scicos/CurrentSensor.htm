Description

Ce bloc est inséré en série dans un circuit électrique pour mesurer le courant qui passe à travers. Si le courant positif entre par le port noir, il est consideré positif. La résistance ohmique de ce composant est zéro.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: oui
  • détection de passage à zéro: non
  • mode: non
  • nombre/taille des entrées régulières: 1 / 1
  • nombre/taille des sorties sorties régulières: 2 / 1 1
  • nombre/taille des entrées évènementielles: 0 /
  • nombre/taille des sorties évènementielles: 0 /
  • possède un état continu: non
  • possède un état discret: non
  • nom de la fonction modelica: CurrentSensor

Fonction d'interface

CurrentSensor.sci

Modèle Modelica 


 class CurrentSensor 
  Pin p;
  Pin n;
  Real i;
 equation
    p.v = n.v;
    p.i = i;
    n.i = -i;
 end CurrentSensor;
