Palette Integer

Module

  • Scicos

Description

Cette palette est dédicacée aux nombres entiers. Des opérateurs variés sont ici implémentés pour la logique et la manipulation des champs de bits ainsi que des portes logiques qui sont fréquemment utilisées dans les circuits numériques.

Blocs

  • BITCLEAR - BITCLEAR Raz un bit
  • BITSET - BITSET Positionne un bit
  • CONVERT - CONVERT Convertion de type de données
  • DFLIPFLOP - DFLIPFLOP Bascule D
  • DLATCH - DLATCH Bascule D
  • EXTRACTBITS - EXTRACTBITS Extraction de bits
  • INTMUL - INTMUL Multiplication matricielle de nombre entiers
  • JKFLIPFLOP - JKFLIPFLOP Bascule JK
  • LOGIC - Logique combinatoire
  • SHIFT - SHIFT Décalage binaire
  • SRFLIPFLOP - SRFLIPFLOP Bascule RS
