create_palette - Générateur de palettes

Module

  • Scicos

Description

Cette fonction génére une palette si Path est une chaîne de caractères indiquant un répertoire où se trouvent des fonctions d'interfaçages de blocs scicos. Si Path est absent ou bien si celui est égal à %t, les palettes standards de Scicos sont construites. Si Path est égal à %f, alors seul le paramètre IntFunc est retourné (le paramètre routines est vide dans ce cas).

Séquence d'appel 

[routines,IntFunc] = create_palette(Path)

Paramètres

  • Path : une chaîne de caractères qui donne le chemein du répertoire contenant des fonctions d'interfaçage de blocs Scicos.
  • routines : un vecteur de chaînes de caractères contenat les noms des fonctions de calcul utilisées dans les blocs Scicos.
  • IntFunc : un vecteur de chaînes de caractères contenant les noms des fonctions d'interfaçage des blocs Scicos.
