Keyboard_shortcuts - Liste des raccourcis clavier

Raccourcis clavier modifiables par l'utilisateur :

  • 'a' : Align
  • 'd' : Delete
  • 'c' : Duplicate
  • 'm' : Move
  • 'z' : Smart Move
  • 'u' : Undo
  • 'f' : Flip
  • 't' : Rotate Left
  • 'T' : Rotate Right
  • 'o' : Open/Set
  • 's' : Save
  • 'i' : Get Info
  • 'r' : Replot
  • 'l' : Link
  • 'q' : Quit
  • '-' : Zoom out
  • '+' : Zoom in
  • '=' : Fit diagram to figure
  • 'g' : Grid
  • 'h' : Help

Raccourcis clavier permanents :

  • 'Ctrl + c' : Copie les objets sélectionnés dans le clipboard scicos
  • 'Ctrl + v' : Colle les objets du clipboard scicos
  • 'Ctrl + a' : Sélectionne tous les objets de la fenêtre courante
  • 'Ctrl + s' : Sauvegarde le diagramme courant
  • 'Ctrl + clic gauche' : Ajoute des objets à la selection courante
