scicos_debug - Définit le niveau de débogage de scicos

Module

  • Scicos

Description

Cette fonction sert à paramétrer le niveau de débogage de la simulation scicos.
Elle peut être utilisée soit en mode "Calc" dans l'éditeur Scicos ou en tant qu'instruction dans un bloc scilab et encore dans une fonction d'interfaçage.

Séquence d'appel 

scicos_debug(level)
level=scicos_debug()

Paramètres

  • level : retourne/définit le niveau de débogage de la simulation scicos.
    • 0 : pas de débogage.
    • 1 : debogage léger et informations affichées dans la console Scilab.
    • 2 : debogage fort : plus d'informations affichées dans la console Scilab et éxécution du bloc "debug" si celui ci est présent dans le diagramme.
    • 3 : debogage silencieux : pas d'informations affichées dans la console Scilab et éxécution du bloc "debug" si celui ci est présent dans le diagramme.
