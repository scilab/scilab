Description

En fournissant un vecteur sur le port d'entrée régulier, ce bloc peut séparer les éléments de l'entrée par des vecteurs sur les ports de sortie. L'entrée est donc définie , où sont numérotés dans un ordre décroissant. Les tailles des ports d'entrée et de sortie sont déterminées par le contexte.

Boîte de dialogue

  • number of output ports or vector de tailles
    un entier positif supérieur ou égale à 8.
    Propriétés : Type 'vec' de taille -1.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: oui
  • détection de passage à zéro: non
  • mode: non
  • entrée régulières:
    - port 1 : taille [0,1] / type 1
  • sorties régulières:
    - port 1 : taille [-1,1] / type 1
    - port 2 : taille [-2,1] / type 1
  • nombre des entrées évènementielles: 0
  • nombre des sorties évènementielles: 0
  • possède un état continu: non
  • possède un état discret: non
  • possède un état objet: non
  • nom de la fonction de calcul: multiplex
