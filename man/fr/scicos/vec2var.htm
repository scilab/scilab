vec2var - Transforme un vecteur de double en une variable scilab

Module

  • Scicos

Description

Les fonctions var2vec / vec2var sont utilisées dans les fonctions d'interfaçages des blocs scilab pour donner la possibilité à l'utilisateur de manipuler des objets scilab avec le registre des paramètres réels (rpar) et avec le registre des états discrets (z).

Séquence d'appel 

[var]=vec2var(vec)

Paramètres

  • vec : Paramètre d'entrée. Un vecteur de nombres réels.
  • var : Paramètre de sortie. Peut-être tous types d'objet scilab.

Exemple 

-->a=list("cos",[1.1,2])
a  = 
a(1)
cos   
a(2)
1.1    2.

2.122-314  
2.122-314  
9.881-324  
1.1

-->c=vec2var(b)
c  =
