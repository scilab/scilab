scicos_block - Définit une structure bloc

Module

  • Scicos

block

Une structure de base qui définit un bloc Scicos.
Cette structure inclut les champs graphics, model, gui et doc.

Size : 5.
Type : scilab list.

  • graphics
    Liste Scilab contenant les informations sur les données graphiques du bloc.

    Taille : 14.
    Type : scilab list.

  • model
    Liste Scilab qui contient les informations du bloc utilisées pour la compilation.

    Taille : 23.
    Type : liste scilab.

  • gui
    Le nom de la fonction d'interface Scilab associée au bloc.

    Taille : 1.
    Type : chaîne de caractères.

  • doc
    Un champ utilisé pour la documentation du bloc.

    Taille : 1.
    Type : chaîne de caractères.
