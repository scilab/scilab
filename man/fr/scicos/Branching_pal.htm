Palette Branching

Module

  • Scicos

Description

Dans la palette Branching, vous pouvez trouver divers blocs pour la gestion des données véhiculées par les liens réguliers.

Blocs

  • CLKFROM - Récepteur de donnée pour un boc CLKGOTO
  • CLKGOTO - Transmetteur de donnée pour un bloc CLKFROM
  • CLKGotoTagVisibility - Définit la portée des données d'un bloc CLKGOTO
  • DEMUX - Démultiplexeur
  • EDGE_TRIGGER - Bloc EDGE TRIGGER
  • ESELECT_f - Bloc synchrone Event-Select
  • Extract_Activation - Bloc Extract_Activation
  • EXTRACTOR - Extractor
  • FROM - FROM Récepteur de données d'un bloc GOTO
  • FROMMO - FROMMO Récepteur de données d'un bloc GOTOMO
  • GOTO - GOTO Transmetteur de données vers un bloc FROM
  • GOTOMO - GOTOMO Transmetteur de données vers un bloc FROMMO
  • GotoTagVisibility - Définit la portée des données d'un bloc GOTO
  • GotoTagVisibilityMO - Définit la portée des données d'un bloc GOTOMO
  • IFTHEL_f - Bloc synchrone If-Then-Else
  • ISELECT_m - Iselect
  • M_SWITCH - Switch à port multiple
  • MUX - Multiplexeur
  • NRMSOM_f - Merge data
  • RELAY_f - Relay
  • SCALAR2VECTOR - Scalaire vers vecteur
  • SELECT_m - Select
  • SWITCH2_m - Switch2
  • SWITCH_f - Switch
